# SEO Strategy Report Examples

This folder contains example content JSON files that demonstrate the complete structure
for different industries.

## Files

### example_content.json
- **Industry**: Hospitality (Hotel)
- **Language**: Spanish
- **Business**: Hotel Castillo de Peñíscola
- **Purpose**: Reference implementation showing all fields populated

This is maintained from the original hotel-seo-strategy-report skill as a complete
working example. While it's hotel-specific, the structure applies to all industries.

## Using Examples

When creating reports for other industries:

1. Copy `example_content.json` as a template
2. Replace hotel-specific sections with industry-appropriate content:
   - Section 7.1: Structural pages (use industry templates from `references/industry-templates.md`)
   - Section 7.2: Experience pages (adapt to industry)
   - Keywords: Use industry-relevant terms
   - Competitors: Research actual competitors
3. Translate section headings if needed (template handles this)
4. Follow the synthesis guide in `references/synthesis-guide.md`
